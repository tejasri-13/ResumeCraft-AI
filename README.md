## 📌 Overview

ResumeCraft AI is a modern web application that helps users build professional resumes quickly and efficiently. The platform provides an intuitive multi-step form, real-time resume preview, dynamic sections, and a resume scoring system to improve resume quality.

This project is built entirely using **HTML, CSS, and JavaScript** without any frameworks.

---

## ✨ Features

* 📄 ATS-Friendly Resume Templates
* 👀 Real-Time Resume Preview
* 📊 Smart Resume Score Analyzer
* 💾 Auto Save using Local Storage
* 🎨 Multiple Theme Options
* ➕ Dynamic Education, Experience, and Projects Sections
* 📱 Fully Responsive Design
* 🖨️ Print Ready Resume Layout
* 🔗 Social Links Integration
* ⚡ Smooth Scrolling Navigation

## 🚀 Key Functionalities

### Multi-Step Resume Builder

* Personal Information
* Education
* Experience
* Skills
* Projects
* Certifications
* Achievements
* Languages & Interests

### Live Resume Generation

Updates the resume instantly as users enter their details.

### Resume Scoring System

Evaluates resume completeness and provides a percentage-based score with improvement suggestions.

### Auto Save

Automatically saves user information in browser Local Storage.

Supports multiple color themes for personalized experience.

## 🏃‍♂️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.

Or use VS Code:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.
