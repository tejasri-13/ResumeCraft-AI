# 🚀 ResumeCraft AI

### Smart ATS Resume Builder

ResumeCraft AI is a modern, responsive resume builder built using HTML, CSS, and JavaScript. It helps users create professional resumes with live preview, smart scoring, multiple themes, and auto-save functionality.

---

## ✨ Features

* 📄 ATS-Friendly Resume Templates
* 📊 Smart Resume Scoring System
* 👀 Real-Time Resume Preview
* 💾 Auto Save using Local Storage
* 🎨 Multiple Themes
* 📱 Fully Responsive Design
* 🖨️ Print and Download Support

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API
